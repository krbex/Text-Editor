# Text Editor

![Github Badge](https://img.shields.io/badge/License-Unlicense-blue.svg)

## Description

A simple text editor that is capable of being downloaded to run offline. This is a single-page PWE application and features data persistence techniques that serve as redundancy in case one option is not supported by the browser.

## Installation

Clone this repository, once cloned run NPM install in the CLI to install all dependencies. The application can be run by using NPM run start which will launch the application to localhost.

## Usage

This project is used to edit text either online or offline
